# Contributing Guidelines

Thanks for your interest in this architecture reference project.  
This repository is not an active codebase but a structured, solution-focused documentation set showcasing:

- AEP → AJO → ACC data flow
- GDPR compliance handling
- Event-based webhook architecture
- Content synchronization pattern
- System-to-system authentication flow

## How to Contribute

If you would like to improve the documentation:

1. Fork the repository  
2. Create a new branch  
3. Add your updates  
4. Submit a pull request  

## Contribution Examples

- Enhancing diagrams  
- Improving workflow explanations  
- Adding missing architectural considerations  
- Refining README or structural clarity  
- Suggesting better GDPR handling patterns  

## Code of Conduct

Please keep contributions respectful, technical, and focused on improving clarity for other readers.
